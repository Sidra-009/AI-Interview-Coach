---

# 🚀 The Solution — AuraCoach

**AuraCoach** is an AI-powered interview simulator that:

- 🧠 Generates interview questions from real job descriptions  
- 🎤 Accepts voice answers (real interview feel)  
- 🔊 Lets you listen to questions  
- ⚡ Provides instant AI feedback  
- 💻 Includes coding interview practice  
- 📊 Tracks improvement across sessions  

It’s like having a personal AI interviewer available 24/7.

---

# ✨ Features

## 🎯 AI Question Generator
- Paste a job description  
- AI generates tailored technical & behavioral questions  

## 🎤 Voice Input (Real Simulation)
- Answer verbally  
- Practice articulation & confidence  

## 🔊 Listen Button
- Hear the question before responding  
- Improves comprehension and clarity  

## ⚡ Instant Feedback
- Structure analysis  
- Clarity & communication scoring  
- Suggestions for improvement  
- Confidence enhancement tips  

## 💻 Coding Interview Mode
- Role-based technical questions  
- Practice explaining your logic clearly  

## 📜 Interview History
- View past sessions  
- Track performance improvements

# 🛠 Tech Stack

- **Frontend:** React + Vite + TypeScript  
- **Backend:** Node.js  
- **Database:** PostgreSQL + Drizzle ORM  
- **Styling:** TailwindCSS + ShadCN UI  
- **AI Integration:** OpenAI API  
- **Audio Handling:** Custom Voice Hooks + Streaming  
- **Deployment:** Replit
